ROCK PAPER SCISSORS
This is a simple Rock Paper Scissors game built in the C programming language. The program allows the user to play against the computer, which randomly selects its move. It is a basic console-based game demonstrating conditional statements, loops, and random number generation in C.

Description :
This project is a basic Rock Paper Scissors game written in C where the player competes against the computer. The computer generates its move randomly, and the program compares both choices to determine the winner. It is a beginner-friendly project that demonstrates key programming concepts such as conditional statements, loops, user input handling, and random number generation using the C standard library. The game runs entirely in the terminal and provides instant results for each round.

#HOW TO COMPILE:

You can compile the Rock Paper Scissors C program using GCC or any standard C compiler.

Using GCC (Recommended) Windows (MinGW) / Linux / Mac

Open your terminal/command prompt and run: gcc rock_paper_scissors.c -o rps

This command compiles the file rock_paper_scissors.c and creates an output executable named rps

Run the Program: ./rps
